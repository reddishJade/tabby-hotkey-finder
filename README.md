# Tabby Hotkey Finder

A plugin for [Tabby](https://github.com/Eugeny/tabby) that helps you find and identify existing hotkey bindings to prevent conflicts. Inspired by the hotkey search functionality in Obsidian.

## Features

- **Key Press Lookup**: Click the "Press key to find" button and hit any key combination on your keyboard to instantly see which hotkeys are using it.
- **Fuzzy Search**: Search for hotkeys by their name, internal ID, or actual key strokes (e.g., search "Ctrl" to see all Ctrl-based shortcuts).
- **Default & Custom Support**: Automatically searches through both Tabby's default built-in hotkeys and your custom overrides.
- **Conflict Prevention**: Easily check if a shortcut you want to use is already taken before you try to bind it.

## Installation

You can install this plugin directly from Tabby's plugin manager by searching for `hotkey-finder`.

## Usage

1. Open Tabby **Settings**.
2. Click on **Hotkey Finder** in the left sidebar.
3. Use the search bar to filter by name/ID, or use the keyboard button to find by physical key press.
